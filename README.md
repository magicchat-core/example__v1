🪄 MagicChat Example — Starter Template
A simple HTML + JS + CSS example to help you quickly integrate MagicChat.io into any website — even if you don't have an existing project yet.

License: MIT

🚀 Overview
This repo is a starter project meant to demonstrate how easily you can embed MagicChat on a static site.

It's perfect if you want to:

🧩 Try out MagicChat without setting up a complex framework

💬 Test your bot or live chat integration locally

⚙️ See how MagicChat behaves on a clean, minimal HTML setup

If you're new to MagicChat or just exploring what it can do — this is the best place to start.

🗂️ Folder Structure
text
example__v1/
├── index.html          # Main demo page
├── style.css           # Basic styling
├── script.js           # Integration logic (MagicChat init, events, etc.)
└── package.json        # Local dev server config
💻 Getting Started
1. Clone this repository

git clone -b starter https://github.com/magicchat-io/example__v1.git


2. Navigate to repo and Install dependencies
cd example__v1
npm install

4. Start the local dev server
npm start
Then open http://localhost:8080 in your browser. You'll see a simple site with MagicChat already integrated.

🧪 Try It Instantly
If you don't have your own app or site yet — no problem! You can use this project as a sandbox to test MagicChat.

Just open index.html, edit the MagicChat script section, and plug in your App ID or API key from the MagicChat Admin Console.
